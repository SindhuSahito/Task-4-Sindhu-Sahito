# Project 4 – Data Visualization using Python

## 📊 Overview
This project focuses on analyzing e-commerce sales data and creating meaningful visualizations to identify business trends, customer behavior, and operational performance.

The project was completed as part of a Data Analytics Internship using Python, Google Colab, and data visualization libraries.

---

## 🎯 Objectives
- Analyze sales and order data  
- Identify top-performing products  
- Study monthly sales trends  
- Understand order status distribution  
- Generate business insights using charts  

---

## 🛠 Tools & Technologies Used
- Python  
- Google Colab  
- Pandas  
- Matplotlib  
- Seaborn  
- Microsoft Excel  
- PowerPoint  

---

## 📁 Dataset Information
The dataset contains:

- 1000+ order records  
- Product details  
- Quantity and pricing information  
- Payment methods  
- Order statuses  
- Coupon codes  
- Referral sources  
- Sales dates (2023–2025)  

---

## 🧹 Data Cleaning Performed
- Removed duplicate records  
- Handled missing values  
- Converted date columns into proper datetime format  
- Created revenue and monthly trend columns  

---

## 📈 Visualizations Created

### 1. Revenue by Product
Bar chart showing products generating the highest revenue.

**Insight:**  
Laptops and monitors contributed the highest share of sales revenue.

---

### 2. Monthly Sales Trend
Line chart showing monthly revenue performance.

**Insight:**  
Sales fluctuated across different months, showing seasonal demand patterns.

---

### 3. Order Status Distribution
Bar/Pie chart showing delivered, cancelled, returned, and pending orders.

**Insight:**  
Most orders were delivered successfully, but cancelled and returned orders require improvement.

---

## 💡 Business Recommendations
- Increase marketing for high-performing products  
- Launch campaigns during low-sales months  
- Improve product quality checks  
- Reduce order cancellations and returns  
- Use data-driven decision making
